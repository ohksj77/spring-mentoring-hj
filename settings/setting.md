## 미리 설치해야하는 프로그램
- docker
- Postman
- Java 17 버전의 jdk
- IntelliJ

## 프로젝트 세팅
1. movie 프로젝트를 인텔리제이에서 열어준다.
2. `.env.template` 파일 내용 및 파일명 `.env`로 수정한다.
3. EnvFile 플러그인을 세팅한다. -> 이미지들을 참고한다.
4. 빌드 방법을 세팅한다. -> 이미지들을 참고한다.
